## Структура каждой иконки (3 слоя)

Каждая SVG-иконка состоит из 3 концентрических кругов:

```
┌─────────────────┐
│  ○ Glow (r=14)  │  opacity: 0.2  — мягкое свечение вокруг
│   ○ Dot (r=8)   │  opacity: 0.75 — основной цветной круг, stroke белый
│    ○ Core (r=3) │  opacity: 0.8  — яркая белая точка в центре
└─────────────────┘
```

**Как создать свою иконку для новой категории:**
```svg
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 32 32" fill="none">
  <circle cx="16" cy="16" r="14" fill="ВАШ_ЦВЕТ" opacity="0.2"/>
  <circle cx="16" cy="16" r="8"  fill="ВАШ_ЦВЕТ" opacity="0.75" stroke="rgba(255,255,255,0.3)" stroke-width="1"/>
  <circle cx="16" cy="16" r="3"  fill="#fff" opacity="0.8"/>
</svg>
```

## Анимации иконок

### 1. Hover (наведение)
При наведении курсора/тапе — иконка увеличивается и становится ярче:

```css
.poi-marker:hover .poi-dot {
  opacity: 1 !important;        /* основной круг — полная яркость */
  stroke-width: 2.5 !important; /* обводка толще */
}
.poi-marker:hover .poi-glow {
  opacity: 0.4 !important;      /* свечение ярче */
}
```

### 2. Pulse (пульсация) — для спецточки (офис)
Расходящееся кольцо вокруг иконки:

```css
@keyframes svgPulse {
  0%   { r: 14; opacity: 0.5; }  /* кольцо маленькое, видимое */
  100% { r: 24; opacity: 0; }    /* кольцо расширяется и исчезает */
}
.office-pulse-svg {
  animation: svgPulse 2s ease-out infinite;
}
```

### 3. Активная точка (в маршруте)
Когда точка добавлена в маршрут:

```
Изменения на .poi-dot:
  stroke      → #BBFF46 (неон-зелёная обводка)
  stroke-width → 2.5
  opacity     → 1

Изменения на .poi-glow:
  opacity     → 0.35 (свечение ярче)

Добавляется:
  • Зелёный круг-бейдж (r=8) с номером маршрута (1, 2, 3...)
  • Текст "START" под первой точкой
  • Эмодзи "🏁" под последней точкой
```

### 4. Маршрутная линия
Пунктирная линия между точками маршрута с анимацией движения:

```css
.route-line {
  stroke: #BBFF46;
  stroke-width: 3;
  stroke-dasharray: 8, 6;
  animation: dashMove 20s linear infinite;
}

@keyframes dashMove {
  to { stroke-dashoffset: -200; }  /* пунктир "бежит" вдоль линии */
}
```

## Поведение карты

### Навигация

| Жест / действие      | Что происходит              |
|----------------------|----------------------------|
| Перетаскивание мышью | Панорамирование (pan)       |
| Колёсико мыши        | Зум к курсору               |
| 1 палец (мобильный)  | Панорамирование             |
| 2 пальца (pinch)     | Зум к центру жеста          |
| Кнопки +/−          | Зум к центру карты          |

### Координатная система

GPS-координаты конвертируются в SVG-координаты:

```
GPS:  lat 7.74..8.18 (юг..север)    lng 98.26..98.42 (запад..восток)
SVG:  viewBox="0 0 600 1000"         Y инвертирован (север = верх)
```

```javascript
function gpsToSvg(lat, lng) {
  var x = ((lng - 98.26) / (98.42 - 98.26)) * 600;
  var y = ((8.18 - lat) / (8.18 - 7.74)) * 1000;
  return { x: x, y: y };
}
```

### Слои карты (порядок отрисовки)

```
1. island-layer   — контур острова (заливка + обводка)
2. areas-layer    — зелёные полупрозрачные овалы районов
3. roads-layer    — линии дорог
4. route-layer    — маршрутная линия
5. poi-layer      — маркеры точек
6. labels-layer   — подписи районов
```

### Стиль элементов карты

**Контур острова:**
```
fill:   rgba(10,60,80,0.5)    — полупрозрачная бирюзовая заливка
stroke: #22cc66, opacity 0.4  — зелёная обводка
```

**Овалы районов:**
```
fill:   rgba(30,140,80,0.15)  — мягкие зелёные области
stroke: rgba(50,200,100,0.08)
```

**Главная дорога (Route 402):**
```
stroke: #BBFF46, opacity 0.35 — жёлто-зелёная пунктирная
stroke-dasharray: 6,4
stroke-width: 2.5
```

**Второстепенные дороги:**
```
stroke: rgba(255,255,255,0.15) — белые тонкие пунктирные
stroke-dasharray: 4,4
stroke-width: 1.5
```

**Подписи районов:**
```
font-family: Inter Tight
font-size: 10px (обычные), 13px (главные — Patong, Phuket Town, Chalong)
fill: rgba(255,255,255,0.3..0.5)
letter-spacing: 2px
text-transform: uppercase
```

## Быстрый старт

1. Скопируй `map-styles.css` в свой проект
2. Добавь HTML-структуру карты (см. `demo.html`)
3. Добавь JS из `../js/map.js` или адаптируй под свой фреймворк
4. Замени данные `PLACES` на свои точки с координатами `lat/lng`
5. Подключи шрифт Inter Tight:
   ```html
   <link href="https://fonts.googleapis.com/css2?family=Inter+Tight:wght@400;500;600;700;800;900&display=swap" rel="stylesheet">
   ```
