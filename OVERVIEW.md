# ThaiGo Lite — Полный обзор проекта

## 1. Общее описание

### 1.1 Зачем существует
ThaiGo Lite — мобильное веб-приложение для аренды байков (скутеров и мотоциклов) на Пхукете, Таиланд. Целевая аудитория — русскоязычные туристы (основная) и международные (EN, DE, FR, ES, TH). Бизнес-модель: прямая аренда с доставкой в отель, без залога паспорта, с договором и страховкой.

### 1.2 Как устроено
SPA (Single Page Application) на ванильном JavaScript без фреймворков. Бандлер — Vite 7.3.1. Деплой на GitHub Pages через GitHub Actions. Максимальная ширина — 430px (mobile-first). Четыре основные «страницы» (табы): Главная, Байки, Гид (маршруты + тест + подбор), Контакты.

### 1.3 Детали реализации
- **Репозиторий**: `amakarovskiy/ThaiGoLite`
- **URL**: `https://amakarovskiy.github.io/ThaiGoLite/`
- **Node.js**: 20
- **Единственная зависимость**: `vite ^7.3.1` (devDependency)
- **Тип модулей**: ESM (`"type": "module"` в package.json)
- **Скрипты**: `dev` (vite), `build` (vite build → dist/), `preview` (vite preview)
- **Vite config**: base `/ThaiGoLite/` для GitHub Actions, иначе `/`; dev-сервер на порту 3000

## 4. Система интернационализации (i18n)

### 4.1 Зачем
Проект ориентирован на международных туристов. Основной язык — русский (RU), плюс EN, DE, FR, ES, TH.

### 4.2 Как работает
- Объект `T` в `i18n.js` содержит все текстовые ключи с переводами на 6 языков
- Функция `t(key)` возвращает текст на текущем языке (fallback → EN → ключ)
- Функция `tpl(key, vars)` — шаблонизатор: `${name}` заменяется на значение
- Определение языка: URL-параметр `?lang=` → localStorage → navigator.language → `'en'`
- При смене языка вызывается `applyTranslations()` — перерисовывает весь UI

### 4.3 Структура LANGS

```js
[
  { code: 'ru', flag: '🇷🇺', label: 'RU', name: 'Русский' },
  { code: 'en', flag: '🇬🇧', label: 'EN', name: 'English' },
  { code: 'de', flag: '🇩🇪', label: 'DE', name: 'Deutsch' },
  { code: 'fr', flag: '🇫🇷', label: 'FR', name: 'Français' },
  { code: 'es', flag: '🇪🇸', label: 'ES', name: 'Español' },
  { code: 'th', flag: '🇹🇭', label: 'TH', name: 'ไทย' }
]
```

### 4.4 Переводы мест
`place-translations.js` содержит объект `PLACE_TR` с переводами `name`, `desc`, `tips` для всех 66 мест. Функции `placeName(place)`, `placeDesc(place)`, `placeTips(place)` проверяют наличие перевода на текущий язык.

### 4.5 Переводы фич байков
Функция `translateFeature(feature, lang)` из `i18n.js` — переводит теги байков (ABS, Keyless, USB и т.д.).

### 4.6 Переводы категорий байков
Объект `BIKE_CAT_TR` — переводит `scooter`, `maxi`, `moto` на все языки.

## 6. Ценообразование (pricing.js)

### 6.1 Сезоны
```
High:     Dec–Mar (месяцы 12, 1, 2, 3)
Shoulder: Apr, Oct, Nov (месяцы 4, 10, 11)
Low:      May–Sep (месяцы 5–9)

Определяется `getCurrentSeason()` по `new Date().getMonth() + 1`.

```
1–2 дня  → tier 0
3–6 дней → tier 1
7–19 дней → tier 2

**Базовая страховка** — бесплатна, включена в стоимость. Покрытие до 30,000 ฿ на госпитализацию.

## 7. Расчёт маршрутов (stats.js)

### 7.1 Константы
- `KM_FACTOR`: 1.35 (множитель на реальное расстояние по дорогам vs по прямой)
- `SPEED_MIN_PER_KM`: 1.5 (мин/км — средняя скорость с учётом остановок)
- `FUEL_L_PER_100KM`: 4 (расход литров на 100 км)
- `FUEL_PRICE_THB`: 43 (цена бензина ฿/литр)
- `TAXI_RATE_PER_KM`: 30 (тариф такси ฿/км — для сравнения)

### 7.2 Алгоритм
1. `haversine(p1, p2)` — расстояние по большому кругу (км)
2. Суммируем расстояния между точками маршрута × KM_FACTOR
3. Время: `km × 1.5` (в минутах)
4. Бензин: `km / 100 × 4 × 43` (в ฿)

### 7.3 Форматирование времени
`formatTime(mins, lang)` — локализованное время:
- RU: `2 ч 30 мин`
- EN: `2 h 30 min`
- TH: `2 ชม. 30 นาที`
